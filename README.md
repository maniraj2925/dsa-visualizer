# DSA Visualizer

An interactive web-based Data Structures and Algorithms Visualizer built using HTML, CSS, and JavaScript. This project helps users understand how popular sorting and searching algorithms work through real-time animations and visual feedback.

## Features

### Sorting Algorithms

* Bubble Sort
* Selection Sort
* Insertion Sort
* Merge Sort
* Quick Sort

### Searching Algorithms

* Binary Search

### Visualization Features

* Dynamic array generation
* Adjustable array size
* Adjustable animation speed
* Real-time algorithm animations
* Comparison and swap counters
* Time complexity display
* Space complexity display
* Interactive dark-themed UI
* Visual state highlighting for algorithm operations

## Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)

## How It Works

### Array Generation

Random arrays are generated with configurable sizes. Each value is represented as a vertical bar whose height is proportional to the value.

### Sorting Visualization

During execution:

* Compared elements are highlighted.
* Swapped elements are animated.
* Sorted elements are marked after completion.
* Statistics are updated in real time.

### Binary Search Visualization

The array is sorted automatically before searching.
The visualizer highlights:

* Left pointer
* Middle pointer
* Right pointer
* Eliminated search regions
* Found target element

## Statistics Tracked

* Algorithm Name
* Number of Comparisons
* Number of Swaps
* Time Complexity
* Space Complexity
